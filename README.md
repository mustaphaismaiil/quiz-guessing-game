# Quiz Quest

An interactive HTML/CSS/JavaScript quiz game that demonstrates functions, arrays, objects, loops, and conditional logic.

## Features

- Multiple-choice quiz using objects and arrays
- Randomized question order each run
- Score tracking and result summary
- Interactive answer selection with correct/incorrect feedback
- Start, next, and restart controls

## Files

- `index.html` – app structure and layout
- `style.css` – visual styling and responsive design
- `script.js` – quiz logic, question flow, and event handling

## Usage

1. Open `index.html` in a browser.
2. Click **Start Quiz**.
3. Select the correct answer for each question.
4. Click **Next Question** to continue.
5. Review your score at the end and click **Try Again** to restart.

## Learning Goals

This project shows how to:

- Use functions to organize behavior
- Store quiz data in arrays and objects
- Iterate through choices with loops
- Apply conditional logic for answer checking
- Update the DOM dynamically with JavaScript

## Notes

The game is built for easy expansion: add new questions to `script.js` or adjust styles in `style.css`.
