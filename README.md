# Quiz Guessing Game

A small web-based quiz guessing game built with HTML, CSS, and JavaScript.

## Project Structure

- `quiz-guessing-game/`
  - `index.html` - the main game page
  - `style.css` - styling for the game interface
  - `script.js` - game logic and interactivity

## How to Run

1. Open `quiz-guessing-game/index.html` in a web browser.
2. Answer the questions presented in the game.

## Notes

- The project is a simple front-end exercise and does not require a server.
- You can customize the questions and appearance by editing `script.js` and `style.css`.
